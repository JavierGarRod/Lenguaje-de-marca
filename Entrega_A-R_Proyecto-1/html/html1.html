¿Qué es HTML?

HTML (HyperText Markup Language) es un lenguaje de marcas para la elaboración de páginas web. Se trata de un estándar soportado y mantenido por el W3C y fue creado en 1991 por Tim Beners Lee. HTML es considerado como el lenguaje web más utilizado y se usa como base para multitud de tecnologías modernas de desarrollo (Angular, Ionic, VUE, React, Symfony, Laravel, etc.).

Existen varias versiones publicadas del Estándar, desde HTML 1 (versión 1) hasta la actual versión HTML 5. En cada una de ellas se agregan y eliminan etiquetas para adaptarse a las necesidades evolutivas de Internet y sus desarrolladores.

En resumen, HTML permite la creación de páginas web básicas, con la inclusión de recursos adicionales como imágenes, vídeos, etc. mediante el uso de etiquetas.
